# Практическая работа 1

Первая работа поднимается в Docker и использует файлы:
- `index.php`
- `init.sql`
- `style.css`

Добавлено только окружение запуска:
- `Dockerfile`
- `docker-compose.yml`

Что есть в работе:
- `Apache + PHP 8.2`
- `MySQL 8.0`
- создание БД `appDB`
- таблица `users`
- тестовая страница с выборкой пользователей
- вывод `phpinfo()`

Запуск:

```bash
docker compose up --build -d
```

Открытие:
- локально на сервере: `http://localhost:8081`
- с другого компьютера: `http://IP_СЕРВЕРА:8081`
